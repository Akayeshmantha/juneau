Class SafeResourceBundle

  • java.lang.Object
    • java.util.ResourceBundle
      • com.ibm.juno.core.utils.SafeResourceBundle
  • Direct Known Subclasses:
    SafeResourceMultiBundle

    public class SafeResourceBundle
extends ResourceBundle
    Wraps a ResourceBundle to gracefully handle missing bundles and entries.

    If the bundle isn't found, getString(key) returns "{!!key}".

    If the key isn't found, getString(key) returns "{!key}".

    Author:
    James Bognar (jbognar@us.ibm.com)
