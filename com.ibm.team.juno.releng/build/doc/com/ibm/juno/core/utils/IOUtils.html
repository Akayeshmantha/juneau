        public static String read(Reader in,
          int length,
          int bufferSize)
                   throws IOException
        Reads the specified input into a String until the end of the input is reached.

        The Reader is automatically closed.

        If the Reader is not an instance of a BufferedReader, then it gets wrapped in a BufferedReader.

        Parameters:
        in - The input reader.
        length - Specify a positive number if the length of the input is known.
        bufferSize - Specify the buffer size to use.
        Returns:

        Throws:
        IOException - If a problem occurred trying to read from the reader.
