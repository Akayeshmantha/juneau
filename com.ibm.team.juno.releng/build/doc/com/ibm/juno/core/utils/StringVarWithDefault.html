    public abstract class StringVarWithDefault
extends StringVar
    Interface for the resolution of string vars with a default value if the resolve(String) method returns null.

    For example, to resolve the system property "myProperty" but resolve to "not found" if the property doesn't exist: "$S{myProperty,not found}"
