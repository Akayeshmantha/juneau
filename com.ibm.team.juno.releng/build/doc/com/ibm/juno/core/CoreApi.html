    public abstract class CoreApi
extends Lockable
    Common super class for all core-API serializers, parsers, and serializer/parser groups.
    Description

    Maintains an inner BeanContextFactory instance that can be used by serializer and parser subclasses to work with beans in a consistent way.

    Provides several duplicate convenience methods from the BeanContextFactory class to set properties on that class from this class.

    Also implements the Lockable interface to allow for easy locking and cloning.

    Author:
    James Bognar (jbognar@us.ibm.com)
