      • BEAN_beanFieldVisibility

        public static final String BEAN_beanFieldVisibility
        Look for bean fields with the specified minimum visibility (Visibility, default=Visibility.PUBLIC).

        Fields are not considered bean properties unless they meet the minimum visibility requirements. For example, if the visibility is PUBLIC and the bean field is protected, then the field will not be interpreted as a bean property.

        Use Visibility.NONE to prevent bean fields from being interpreted as bean properties altogether.

      • BEAN_addNotBeanPackages

        public static final String BEAN_addNotBeanPackages
        Add to the list of packages whose classes should not be considered beans (String, comma-delimited list).

        When specified, the current list of ignore packages are appended to. The default list of ignore packages are as follows:

        • java.lang
        • java.lang.annotation
        • java.lang.ref
        • java.lang.reflect
        • java.io
        • java.net
        • java.nio.*
        • java.util.*
        Any classes within these packages will be serialized to strings using Object.toString().

        Note that you can specify prefix patterns to include all subpackages.
