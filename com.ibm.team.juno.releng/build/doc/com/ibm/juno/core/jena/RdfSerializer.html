    @Produces(value="text/xml+rdf+abbrev",
          contentType="text/xml+rdf")
public class RdfSerializer
extends WriterSerializer
    Serializes POJOs to RDF.
    Configurable properties

    Refer to Configurable Properties for the entire list of configurable properties.

    Behavior-specific subclasses

    The following direct subclasses are provided for language-specific serializers:

    • RdfSerializer.Xml - RDF/XML.
    • RdfSerializer.XmlAbbrev - RDF/XML-ABBREV.
    • RdfSerializer.NTriple - N-TRIPLE.
    • RdfSerializer.Turtle - TURTLE.
    • RdfSerializer.N3 - N3.
    Additional Information

    See RDF Overview for an overview of RDF support in Juno.

    Author:
    James Bognar (jbognar@us.ibm.com)
