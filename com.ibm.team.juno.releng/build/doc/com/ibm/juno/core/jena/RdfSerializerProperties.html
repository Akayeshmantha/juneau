      • RDF_addRootProperty

        public static final String RDF_addRootProperty
        Add RDF root identifier property to root node (Boolean, default=false).

        When enabled an RDF property http://www.ibm.com/juno/root is added with a value of "true" to identify the root node in the graph. This helps locate the root node during parsing.

        If disabled, the parser has to search through the model to find any resources without incoming predicates to identify root notes, which can introduce a considerable performance degradation.
