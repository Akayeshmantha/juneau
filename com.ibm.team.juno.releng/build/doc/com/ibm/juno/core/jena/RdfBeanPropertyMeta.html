      • getNamespace

        public Namespace getNamespace()
        Returns the RDF namespace associated with this bean property.

        Namespace is determined in the following order:

        1. Rdf.prefix() annotation defined on bean property field.
        2. Rdf.prefix() annotation defined on bean getter.
        3. Rdf.prefix() annotation defined on bean setter.
        4. Rdf.prefix() annotation defined on bean.
        5. Rdf.prefix() annotation defined on bean package.
        6. Rdf.prefix() annotation defined on bean superclasses.
        7. Rdf.prefix() annotation defined on bean superclass packages.
        8. Rdf.prefix() annotation defined on bean interfaces.
        9. Rdf.prefix() annotation defined on bean interface packages.
        Returns:
        The namespace associated with this bean property, or null if no namespace is associated with it.
