com.ibm.juno.core.parser

Class ParserListener

  • java.lang.Object
    • com.ibm.juno.core.parser.ParserListener

  • public class ParserListener
extends Object
    Class for listening for certain parse events during a document parse.

    Listeners can be registered with parsers through the Parser.addListener(ParserListener) method.

    It should be noted that listeners are not automatically copied over to new parsers when a parser is cloned.
    Author:
    James Bognar (jbognar@us.ibm.com)

        public <T> void onUnknownProperty(String propertyName,
                         Class<T> beanClass,
                         T bean,
                         int line,
                         int col)
        Gets called when an unknown bean property is detected in a document.

        This method only gets called if BeanContextProperties.BEAN_ignoreUnknownBeanProperties setting is true. Otherwise, the parser will throw a ParseException.

        Type Parameters:
        T - The class type of the bean.
        Parameters:
        propertyName - The property name encountered in the document.
        beanClass - The bean class.
        bean - The bean.
        line - The line number where the unknown property was found (-1 if parser doesn't support line/column indicators).
        col - The column number where the unknown property was found (-1 if parser doesn't support line/column indicators).
