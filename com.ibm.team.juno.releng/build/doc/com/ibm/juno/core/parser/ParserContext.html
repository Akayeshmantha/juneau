com.ibm.juno.core.parser

Class ParserContext

  • java.lang.Object
    • com.ibm.juno.core.parser.ParserContext
  • Direct Known Subclasses:
    HtmlParserContext, JsonParserContext, RdfParserContext, UonParserContext, XmlParserContext

    public class ParserContext
extends Object
    Context object that lives for the duration of a single parsing of Parser and its subclasses.

    This class is NOT thread safe. It is meant to be discarded after one-time use.

    Author:
    James Bognar (jbognar@us.ibm.com)
