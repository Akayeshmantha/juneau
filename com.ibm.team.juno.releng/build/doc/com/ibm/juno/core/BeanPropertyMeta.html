com.ibm.juno.core

Class BeanPropertyMeta<T>

  • java.lang.Object
    • com.ibm.juno.core.BeanPropertyMeta<T>
  • Type Parameters:
    T - The class type of the bean that this metadata applies to.

    public class BeanPropertyMeta<T>
extends Object
    Contains metadata about a bean property.

    Contains information such as type of property (e.g. field/getter/setter), class type of property value, and whether any filters are associated with this property.

    Developers will typically not need access to this class. The information provided by it is already exposed through several methods on the BeanMap API.

    Author:
    James Bognar (jbognar@us.ibm.com)
