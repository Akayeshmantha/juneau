      • getNamespace

        public Namespace getNamespace()
        Returns the XML namespace associated with this bean property.

        Namespace is determined in the following order:

        1. Xml.prefix() annotation defined on bean property field.
        2. Xml.prefix() annotation defined on bean getter.
        3. Xml.prefix() annotation defined on bean setter.
        4. Xml.prefix() annotation defined on bean.
        5. Xml.prefix() annotation defined on bean package.
        6. Xml.prefix() annotation defined on bean superclasses.
        7. Xml.prefix() annotation defined on bean superclass packages.
        8. Xml.prefix() annotation defined on bean interfaces.
        9. Xml.prefix() annotation defined on bean interface packages.
        Returns:
        The namespace associated with this bean property, or null if no namespace is associated with it.
