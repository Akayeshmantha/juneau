      • XML_addJsonStringTypeAttrs

        public static final String XML_addJsonStringTypeAttrs
        Add JSON type attributes for strings to output (Boolean, default=false).

        If true, type attributes will be added to elements in the XML for string nodes.

        By default, these attributes are not added, and the parser will assume that the content type of the node is string by default.

        This feature is disabled if XML_addJsonTypeAttrs is disabled.

      • XML_autoDetectNamespaces

        public static final String XML_autoDetectNamespaces
        Auto-detect namespace usage (Boolean, default=true).

        Detect namespace usage before serialization.

        Used in conjunction with XML_addNamespaceUrisToRoot to reduce the list of namespace URLs appended to the root element to only those that will be used in the resulting document.

        If enabled, then the data structure will first be crawled looking for namespaces that will be encountered before the root element is serialized.

        This setting is ignored if XML_enableNamespaces is not enabled.

        IMPORTANT NOTE: Auto-detection of namespaces can be costly performance-wise. In high-performance environments, it's recommended that namespace detection be disabled, and that namespaces be manually defined through the XML_namespaces property.

        See Also:
