Class NamespaceFactory

  • java.lang.Object
    • com.ibm.juno.core.xml.NamespaceFactory

  • public final class NamespaceFactory
extends Object
    Factory class for getting unique instances of Namespace objects.

    For performance reasons, Namespace objects are stored in IdentityLists. For this to work property, namespaces with the same name and URI must only be represented by a single Namespace instance. This factory class ensures this identity uniqueness.

    Author:
    James Bognar (jbognar@us.ibm.com)
