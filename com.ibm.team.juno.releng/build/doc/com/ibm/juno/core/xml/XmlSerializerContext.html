Class XmlSerializerContext

  • java.lang.Object
    • com.ibm.juno.core.serializer.SerializerContext
      • com.ibm.juno.core.xml.XmlSerializerContext
  • Direct Known Subclasses:
    HtmlSerializerContext, RdfSerializerContext

    public class XmlSerializerContext
extends SerializerContext
    Context object that lives for the duration of a single serialization of the XmlSerializer.

    See SerializerContext for details.

    This class is NOT thread safe. It is meant to be discarded after one-time use.

    Author:
    James Bognar (jbognar@us.ibm.com)
