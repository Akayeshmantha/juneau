      • XML_preserveRootElement

        public static final String XML_preserveRootElement
        Preserve root element during generalized parsing (Boolean, default=false).

        If true, when parsing into a generic ObjectMap, the map will contain a single entry whose key is the root element name. Example:

        XML ObjectMap.toString(), preserveRootElement==false ObjectMap.toString(), preserveRootElement==true
        <root><a>foobar</a></root> { a:'foobar' } { root: { a:'foobar' }}
