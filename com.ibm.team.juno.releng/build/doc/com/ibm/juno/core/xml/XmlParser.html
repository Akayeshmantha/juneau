    @Consumes(value={"text/xml","application/xml"})
public class XmlParser
extends ReaderParser
    Parses text generated by the XmlSerializer class back into a POJO model.
    Media types

    Handles Content-Type types: text/xml

    Description

    See the XmlSerializer class for a description of Juno-generated XML.
