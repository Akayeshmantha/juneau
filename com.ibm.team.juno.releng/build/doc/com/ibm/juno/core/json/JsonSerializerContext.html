Class JsonSerializerContext

  • java.lang.Object
    • com.ibm.juno.core.serializer.SerializerContext
      • com.ibm.juno.core.json.JsonSerializerContext

  • public final class JsonSerializerContext
extends SerializerContext
    Context object that lives for the duration of a single serialization of JsonSerializer and its subclasses.

    See SerializerContext for details.

    This class is NOT thread safe. It is meant to be discarded after one-time use.
