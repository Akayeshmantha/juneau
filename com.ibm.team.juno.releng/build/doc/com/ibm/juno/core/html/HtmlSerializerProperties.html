    • Field Detail

      • HTML_uriAnchorText

        public static final String HTML_uriAnchorText
        Anchor text source (String, default=TO_STRING).

        When creating anchor tags (e.g. <a href='...'>text</a>) in HTML, this setting defines what to set the inner text to.

        Possible values:

        • TO_STRING / "toString" - Set to whatever is returned by Object.toString() on the object.
        • URI / "uri" - Set to the URI value.
        • LAST_TOKEN / "lastToken" - Set to the last token of the URI value.
        • PROPERTY_NAME / "propertyName" - Set to the bean property name.
