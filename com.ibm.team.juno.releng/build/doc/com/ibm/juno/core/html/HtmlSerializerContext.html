Class HtmlSerializerContext

  • public final class HtmlSerializerContext
extends XmlSerializerContext
    Context object that lives for the duration of a single serialization of HtmlSerializer and its subclasses.

    See SerializerContext for details.

    This class is NOT thread safe. It is meant to be discarded after one-time use.

    Author:
    James Bognar (jbognar@us.ibm.com)
