    public class ReaderFilter
extends PojoFilter<Reader,Object>
    Transforms the contents of a Reader into an Object.
    Description

    The Reader must contain JSON, Juno-generated XML (output from XmlSerializer), or Juno-generated HTML (output from JsonSerializer) in order to be parsed correctly.

    Useful for serializing models that contain Readers created by RestCall instances.

    This is a one-way filter, since Readers cannot be reconstituted.

    Behavior-specific subclasses

    The following direct subclasses are provided for convenience:

    • ReaderFilter.Json - Parses JSON text.
    • ReaderFilter.Xml - Parses XML text.
    • ReaderFilter.Html - Parses HTML text.
    • ReaderFilter.PlainText - Parses plain text.
