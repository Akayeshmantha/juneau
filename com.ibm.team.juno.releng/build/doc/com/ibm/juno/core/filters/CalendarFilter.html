    public class CalendarFilter
extends PojoFilter<Calendar,String>
    Transforms Calendars to Strings.
    Behavior-specific subclasses

    The following direct subclasses are provided for convenience:

    • CalendarFilter.ToString - Transforms to Strings using the Date.toString() method.
    • CalendarFilter.ISO8601DT - Transforms to ISO8601 date-time strings.
    • CalendarFilter.ISO8601DTZ - Same as CalendarFilter.ISO8601DT, except always serializes in GMT.
    • CalendarFilter.RFC2822DT - Transforms to RFC2822 date-time strings.
    • CalendarFilter.RFC2822DTZ - Same as CalendarFilter.RFC2822DT, except always serializes in GMT.
    • CalendarFilter.RFC2822D - Transforms to RFC2822 date strings.
    • CalendarFilter.Simple - Transforms to simple "yyyy/MM/dd HH:mm:ss" strings.
    • CalendarFilter.Medium - Transforms to DateFormat.MEDIUM strings.
