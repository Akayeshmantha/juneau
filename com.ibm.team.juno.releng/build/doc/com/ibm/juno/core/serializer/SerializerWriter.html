Class SerializerWriter

  • java.lang.Object
    • java.io.Writer
      • com.ibm.juno.core.serializer.SerializerWriter
  • All Implemented Interfaces:
    Closeable, Flushable, Appendable, AutoCloseable
    Direct Known Subclasses:
    JsonSerializerWriter, UonSerializerWriter, XmlSerializerWriter

    public class SerializerWriter
extends Writer
    Simple wrapper around a standard Writer with additional methods.

    Modeled after the Java ProcessBuilder class so that you can chain commands to reduce the need for string concatenation for performance reasons.

    Example

    writer.append("foo").nl().i(5).append("bar");

    Author:
    James Bognar (jbognar@us.ibm.com)

      • appendUri

        public SerializerWriter appendUri(Object uri)
                           throws IOException
        Appends the specified object as a URI.

        Object is converted to a String using toString(), so this will work on URL or URI objects, or any other type that returns a URI via it's toString() method.

        If the URI is relative (i.e. without a schema and not prepended with '/') the URI will be prepended with absolutePathUriBase and relativeUriBase.

        If the URI is context-absolute (i.e. without a schema, but prepended with '/') the URI will be prepended with absolutePathUriBase.

        Parameters:
        uri - The URI to serialize.
        Returns:
        This object (for method chaining).

        IOException - If a problem occurred trying to write to the writer.
