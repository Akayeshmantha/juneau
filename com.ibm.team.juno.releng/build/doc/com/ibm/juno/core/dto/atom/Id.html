Class Id

    • com.ibm.juno.core.dto.atom.Common

  • @Xml(name="id")
public class Id
extends Common
    Represents an atomId construct in the RFC4287 specification.

    Schema

    atomId = element atom:id { atomCommonAttributes, (atomUri) }

    Refer to com.ibm.juno.core.dto.atom for further information about ATOM support.

    James Bognar (jbognar@us.ibm.com)
