Class UonParserContext

  • java.lang.Object
    • com.ibm.juno.core.parser.ParserContext
      • com.ibm.juno.core.urlencoding.UonParserContext

  • public class UonParserContext
extends ParserContext
    Context object that lives for the duration of a single parsing in UonParser and UrlEncodingParser.

    This class is NOT thread safe. It is meant to be discarded after one-time use.

        Create a specialized parser context for parsing URL parameters.

        The main difference is that characters are never decoded, and the UonParserProperties.UON_decodeChars property is always ignored.

        Parameters:
        beanContext - The bean context being used.
        pp - The default parser properties.
        upp - The default UON-Encoding properties.
