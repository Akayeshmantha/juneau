com.ibm.juno.core.annotation

Annotation Type Consumes


  • @Documented
@Target(value=TYPE)
@Retention(value=RUNTIME)
@Inherited
public @interface Consumes
    Annotation used on subclasses of Parser to identify the media types that it consumes.
    Description

    Provides a way to define the contents of Parser.getMediaTypes() through an annotation.

    The Parser.getMediaTypes() default implementation gathers the media types by looking for this annotation. It should be noted that this annotation is optional and that the Parser.getMediaTypes() method can be overridden by subclasses to return the media types programmatically.

    Examples

    Standard example:

    @Consumes({"application/json","text/json"}) public class JsonParser extends ReaderParser {...}

    The media types can also be media-range values per RFC2616/14.1.

    // Consumes any text @Consumes({"text\/*"}) public class AnythingParser extends ReaderParser {...} // Consumes anything @Consumes({"*\/*"}) public class AnythingParser extends ReaderParser {...}
