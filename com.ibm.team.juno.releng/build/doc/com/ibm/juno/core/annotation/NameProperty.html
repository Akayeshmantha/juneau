com.ibm.juno.core.annotation

Annotation Type NameProperty


  • @Target(value=METHOD)
@Retention(value=RUNTIME)
@Inherited
public @interface NameProperty
    Identifies a setter as a method for setting the name of a POJO as it's known by its parent object.

    For example, the Section class must know the name it's known by it's parent ConfigFileImpl class, so parsers will call this method with the sectio name using the Section.setName(String) method.

    Author:
    James Bognar (jbognar@us.ibm.com)
