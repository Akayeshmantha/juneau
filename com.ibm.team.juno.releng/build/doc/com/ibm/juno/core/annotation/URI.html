com.ibm.juno.core.annotation

Annotation Type URI


  • @Documented
@Target(value={TYPE,FIELD,METHOD})
@Retention(value=RUNTIME)
@Inherited
public @interface URI
    Used to identify a class or bean property as a URI.

    By default, instances of URL and URI are considered URIs during serialization, and are handled differently depending on the serializer (e.g. HtmlSerializer creates a hyperlink, RdfXmlSerializer creates an rdf:resource object, etc...).

    This annotation allows you to identify other classes that return URIs via toString() as URI objects.

    Relative URIs are automatically prepended with SerializerProperties.SERIALIZER_absolutePathUriBase and SerializerProperties.SERIALIZER_relativeUriBase during serialization just like relative URIs.

    This annotation can be applied to classes, interfaces, or bean property methods for fields.

    Examples

    // Applied to a class whose toString() method returns a URI. @URI public class MyURI { @Override public String toString() { return "http://localhost:9080/foo/bar"; } } // Applied to bean properties public class MyBean { @URI public String beanUri; @URI public String getParentUri() { ... } }
