com.ibm.juno.core.annotation

Annotation Type Produces


  • @Documented
@Target(value=TYPE)
@Retention(value=RUNTIME)
@Inherited
public @interface Produces
    Annotation used on subclasses of Serializer to identify the media types that it produces.
    Description

    Provides a way to define the contents of Serializer.getMediaTypes() through an annotation.

    The Serializer.getMediaTypes() default implementation gathers the media types by looking for this annotation. It should be noted that this annotation is optional and that the Serializer.getMediaTypes() method can be overridden by subclasses to return the media types programmatically.

    Examples

    Standard example:

    @Produces({"application/json","text/json"}) public class JsonSerializer extends WriterSerializer {...}

    The media types can also be media-range values per RFC2616/14.1. When meta-characters are used, you should specify the contentType() value to indicate the real media type value that can be set on the Content-Type response header.

    // Produces any text @Produces(value="text\/*", contentType="text/plain") public class AnythingSerializer extends WriterSerializer {...} // Produces anything @Produces(value="*\/*", contentType="text/plain") public class AnythingSerializer extends WriterSerializer {...}
