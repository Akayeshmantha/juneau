Class RestCallInterceptor

  • java.lang.Object
    • com.ibm.juno.client.RestCallInterceptor
  • Direct Known Subclasses:
    RestCallLogger

    public abstract class RestCallInterceptor
extends Object
    Used to intercept http connection responses to allow modification of that response before processing and for listening for call lifecycle events.

    Useful if you want to prevent RestCallExceptions from being thrown on error conditions.
