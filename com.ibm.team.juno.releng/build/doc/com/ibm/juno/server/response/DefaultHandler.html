    public class DefaultHandler
extends Object
implements ResponseHandler
    Response handler for POJOs not handled by other handlers.

    This uses the serializers defined on the response to serialize the POJO.

    The Serializer used is based on the Accept header on the request.

    The Content-Type header is set to the mime-type defined on the selected serializer based on the @Produces.contentType annotation.

    This handler is registered by default on RestServlets via the default implementation of the RestServlet.createResponseHandlers() method.
