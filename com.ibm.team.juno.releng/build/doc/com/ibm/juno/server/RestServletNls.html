Class RestServletNls

  • java.lang.Object
    • com.ibm.juno.server.RestServletNls

  • public final class RestServletNls
extends Object
    Utility class for extracting messages from resource bundles identified through the RestResource.messages() annotation.

    When mutiple servlets are using the same resource bundle, the key names can be prefixed with the servlet class name to prevent name conflicts.

    Author:
    James Bognar (jbognar@us.ibm.com)
