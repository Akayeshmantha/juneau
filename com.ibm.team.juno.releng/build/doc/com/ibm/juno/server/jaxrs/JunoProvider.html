com.ibm.juno.server.jaxrs

Annotation Type JunoProvider


  • @Documented
@Target(value=TYPE)
@Retention(value=RUNTIME)
@Inherited
public @interface JunoProvider
    Annotations applicable to subclasses of BaseProvider.
    Description

    Used to associate serializers, parsers, filters, and properties with instances of BaseProvider.

    Author:
    James Bognar (jbognar@us.ibm.com)

    • Optional Element Summary

      Optional Elements 
      Modifier and Type Optional Element and Description
      Class<?>[] filters
      Provider-level POJO filters.
      Class<? extends Parser<?>>[] parsers
      Specifies a list of Parser classes to add to the list of parsers available for this provider.
      Property[] properties
      Provider-level properties.

    • Element Detail

      • filters

        public abstract Class<?>[] filters
        Provider-level POJO filters.

        These filters are applied to all serializers and parsers being used by the provider.

        If the specified class is an instance of Filter, then that filter is added. Any other classes are wrapped in a BeanFilter to indicate that subclasses should be treated as the specified class type.

        Default:
        {}

      • properties

        public abstract Property[] properties
        Provider-level properties.

        Any of the following property names can be specified:

        • RestServletProperties
        • BeanContextProperties
        • SerializerProperties
        • ParserProperties
        • JsonSerializerProperties
        • XmlSerializerProperties
        • XmlParserProperties

        Property values will be converted to the appropriate type.

        These properties can be augmented/overridden through the RestMethod.properties() annotation on the REST method.

        Default:
        {}

      • serializers

        public abstract Class<? extends Serializer<?>>[] serializers
        Specifies a list of Serializer classes to add to the list of serializers available for this provider.

        This annotation can only be used on Serializer classes that have no-arg constructors.

        Default:
        {}

      • parsers

        public abstract Class<? extends Parser<?>>[] parsers
        Specifies a list of Parser classes to add to the list of parsers available for this provider.

        This annotation can only be used on Parser classes that have no-arg constructors.

        Default:
        {}
