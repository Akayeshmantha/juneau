Package com.ibm.juno.server.annotation

REST servlet annotations

See: Description

  • Class Summary 
    Class Description
    VarCategory
    Predefined string constants for the Var.category() annotation.
  • Enum Summary 
    Enum Description
    Inherit
    Inheritance values for the RestMethod.serializersInherit() and RestMethod.parsersInherit() annotations.
  • Annotation Types Summary 
    Annotation Type Description
    Attr
    Annotation that can be applied to a parameter of a RestMethod annotated method to identify it as a variable in a URL path pattern converted to a POJO.
    Content
    Annotation that can be applied to a parameter of a RestMethod annotated method to identify it as the HTTP request body converted to a POJO.
    HasParam
    Annotation that can be applied to a parameter of a RestMethod annotated method to identify whether or not the request has the specified GET or POST parameter.
    HasQParam
    Identical to @HasParam, but only checks the existing of the parameter in the URL string, not URL-encoded form posts.
    Header
    Annotation that can be applied to a parameter of a RestMethod annotated method to identify it as a HTTP request header converted to a POJO.
    Messages
    Annotation that can be applied to a parameter of a RestMethod annotated method to identify it as the resource bundle for the request locale.
    Method
    Annotation that can be applied to a parameter of a RestMethod annotated method to identify it as the HTTP method.
    Param
    Annotation that can be applied to a parameter of a RestMethod annotated method to identify it as a URL query parameter converted to a POJO.
    PathRemainder
    Annotation that can be applied to a parameter of a RestMethod annotated method to identify it as the URL parameter remainder after a path pattern match.
    Properties
    Annotation that can be applied to a parameter of a RestMethod annotated method to identify the request-duration properties object for the current request.
    Property
    Property name/value pair used in the RestResource.properties() annotation.
    QParam
    Identical to @Param, but only retrieves the parameter from the URL string, not URL-encoded form posts.
    Response
    Annotation used in conjunction with RestMethod.responses() to identify possible responses by the method.
    RestMethod
    Identifies a REST Java method on a RestServlet implementation class.
    RestResource
    Optionally used to associate metadata on an instance of RestServlet.
    Var
    Annotation used in conjunction with RestMethod.input() and Response.output() to identify content and header descriptions on specific method responses.
