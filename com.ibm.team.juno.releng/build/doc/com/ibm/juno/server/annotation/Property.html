com.ibm.juno.server.annotation

Annotation Type Property


  • @Documented
@Target(value=ANNOTATION_TYPE)
@Retention(value=RUNTIME)
@Inherited
public @interface Property
    Property name/value pair used in the RestResource.properties() annotation.

    Any of the following property names can be specified:

    • BeanContextProperties
    • SerializerProperties
    • ParserProperties
    • JsonSerializerProperties
    • RdfSerializerProperties
    • RdfParserProperties
    • RdfProperties
    • XmlSerializerProperties
    • XmlParserProperties

    Property values types that are not Strings will automatically be converted to the correct type (e.g. Boolean, etc...).

    See RestResource.properties() for more information.
