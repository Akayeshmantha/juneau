028/**
029 * Core class of the Juno architecture.
030 * <p>
031 *    This class servers multiple purposes:
032 *    <ul>
033 *       <li>Provides the ability to wrap beans inside {@link Map} interfaces.
034 *       <li>Serves as a repository for metadata on POJOs, such as associated {@link Filter filters}, {@link PropertyNamer property namers}, etc...
035 *          which are used to tailor how POJOs are serialized and parsed.
036 *       <li>Serves as a common utility class for all {@link Serializer Serializers} and {@link Parser Parsers}
037 *             for serializing and parsing Java beans.
038 *    </ul>
039 *
040 *
041 * <h5 class='topic'>Bean Contexts</h5>
042 * <p>
043 *    Typically, it will be sufficient to use the existing {@link #DEFAULT} contexts for creating
044 *    bean maps.  However, if you want to tweak any of the settings on the context, you must
045 *    either clone the default context or create a new one from scratch (whichever is simpler for you).
046 *    You'll notice that this context class uses a fluent interface for defining settings.
047 * <p>
048 *    Bean context factories can be locked using the {@link BeanContextFactory#lock()} method.  This makes the context settings
049 *    read-only.  Attempting to change a setting on a locked context will cause a {@link LockedException}
050 *    to be thrown.  The default context is locked by default.  Cloning a locked context using the
051 *    {@link #clone()} method produces a new unlocked context.  Locking a context is optional, although
052 *    it can prevent errors where bean contexts, parsers, or serializers start behaving differently
053 *    because a setting was changed.  As a general rule, if you want to change a setting on an existing
054 *    context, you should clone it, modify the setting(s), and then lock the new context.
055 *
056 *
057 * <h5 class='topic'>BeanContext settings</h5>
058 *    <code>BeanContexts</code> have several settings that can be used to tweak behavior on how beans are handled.
059 * <p>
060 *    Some settings (e.g. {@link BeanContextProperties#BEAN_beansRequireDefaultConstructor}) are used to differentiate between bean and non-bean classes.
061 *    Attempting to create a bean map around one of these objects will throw a {@link BeanRuntimeException}.
062 *    The purpose for this behavior is so that the serializers can identify these non-bean classes and convert them to plain strings using the {@link Object#toString()} method.
063 * <p>
064 *    Some settings (e.g. {@link BeanContextProperties#BEAN_beanFieldVisibility}) are used to determine what kinds of properties are detected on beans.
065 * <p>
066 *    Some settings (e.g. {@link BeanContextProperties#BEAN_beanMapPutReturnsOldValue}) change the runtime behavior of bean maps.
067 *
068 *
069 * <h6 class='topic'>Configurable properties</h6>
070 *    This class has configurable properties that can be set through the {@link BeanContextFactory#setProperty(String, Object)} method.
071 * <p>
072 *    See {@link BeanContextProperties} for settings applicable to this class.
073 *
074 *
075 * <h6 class='topic'>Examples</h6>
076 * <p class='bcode'>
077 *    <jc>// Construct a context from scratch.</jc>
078 *    BeanContext beanContext = <jk>new</jk> BeanContextFactory()
079 *       .setProperty(BeanContextProperties.<jsf>BEAN_beansRequireDefaultConstructor</jsf>, <jk>true</jk>)
080 *       .addNotBeanClasses(Foo.<jk>class</jk>)
081 *       .getBeanContext();
082 *
083 *    <jc>// Clone an existing context factory.</jc>
084 *    BeanContext beanContext = otherBeanContextFactory.clone()
085 *       .setProperty(BeanContextProperties.<jsf>BEAN_beansRequireDefaultConstructor</jsf>, <jk>true</jk>)
086 *       .addNotBeanClasses(Foo.<jk>class</jk>)
087 *       .getBeanContext();
088 * </p>
089 *
090 *
091 * <h5 class='topic'>Bean Maps</h5>
092 * <p>
093 *    {@link BeanMap BeanMaps} are wrappers around Java beans that allow properties to be retrieved and
094 *    set using the common {@link Map#put(Object,Object)} and {@link Map#get(Object)} methods.<br>
095 *    <br>
096 *    Bean maps are created in two ways...
097 *    <ol>
098 *       <li> {@link BeanContext#forBean(Object) BeanContext.forBean()} - Wraps an existing bean inside a {@code Map} wrapper.
099 *       <li> {@link BeanContext#newBeanMap(Class) BeanContext.newInstance()} - Create a new bean instance wrapped in a {@code Map} wrapper.
100 *    </ol>
101 *
102 *
103 * <h6 class='topic'>Examples</h6>
104 * <p class='bcode'>
105 *    <jc>// A sample bean class</jc>
106 *    <jk>public class</jk> Person {
107 *       <jk>public</jk> String getName();
108 *       <jk>public void</jk> setName(String name);
109 *       <jk>public int</jk> getAge();
110 *       <jk>public void</jk> setAge(<jk>int</jk> age);
111 *    }
112 *
113 *    <jc>// Wrap an existing bean in a new bean map</jc>
114 *    BeanMap&lt;Person&gt; m1 = BeanContext.<jsf>DEFAULT</jsf>.forBean(<jk>new</jk> Person());
115 *    m1.put(<js>"name"</js>, <js>"John Smith"</js>);
116 *    m1.put(<js>"age"</js>, 45);
117 *
118 *    <jc>// Create a new bean instance wrapped in a new bean map</jc>
119 *    BeanMap&lt;Person&gt; m2 = BeanContext.<jsf>DEFAULT</jsf>.newInstance(Person.<jk>class</jk>);
120 *    m2.put(<js>"name"</js>, <js>"John Smith"</js>);
121 *    m2.put(<js>"age"</js>, 45);
122 *    Person p = m2.getBean();  <jc>// Get the bean instance that was created.</jc>
123 * </p>
124 *
125 *
126 * <h5 class='topic'>Bean Annotations</h5>
127 * <p>
128 *    This package contains annotations that can be applied to
129 *    class definitions to override what properties are detected on a bean.
130 * <h6 class='topic'>Examples</h6>
131 * <p class='bcode'>
132 *    <jc>// Bean class definition where only property 'name' is detected.</jc>
133 *    <ja>&#64;Bean</ja>(properties={<js>"name"</js>})
134 *    <jk>public class</jk> Person {
135 *       <jk>public</jk> String getName();
136 *       <jk>public void</jk> setName(String name);
137 *       <jk>public int</jk> getAge();
138 *       <jk>public void</jk> setAge(<jk>int</jk> age);
139 *    }
140 * <p>
141 *    See {@link Bean @Bean} and {@link BeanProperty @BeanProperty} for more information.
142 *
143 *
144 * <h5 class='topic'>Beans with read-only properties</h5>
145 * <p>
146 *    Bean maps can also be defined on top of beans with read-only properties by adding a
147 *    {@link BeanConstructor @BeanConstructor} annotation to one of the constructors on the
148 *    bean class.  This will allow read-only properties to be set through constructor arguments.
149 * <p>
150 *    When the <code>@BeanConstructor</code> annotation is present, bean instantiation is delayed until the call to {@link BeanMap#getBean()}.
151 *    Until then, bean property values are stored in a local cache until <code>getBean()</code> is called.
152 *    Because of this additional caching step, parsing into read-only beans tends to be slower and use
153 *    more memory than parsing into beans with writable properties.
154 * <p>
155 *    Attempting to call {@link BeanMap#put(String,Object)} on a read-only property after calling {@link BeanMap#getBean()}
156 *    will result in a {@link BeanRuntimeException} being thrown.
157 *    Multiple calls to {@link BeanMap#getBean()} will return the same bean instance.
158 * <p>
159 *    Beans can be defined with a combination of read-only and read-write properties.
160 * <p>
161 *    See {@link BeanConstructor @BeanConstructor} for more information.
162 *
163 *
164 * <h5 class='topic'>Filters</h5>
165 * <p>
166 *    {@link Filter Filters} are used to tailor how beans and non-beans are handled.<br>
167 *    There are two subclasses of filters:
168 *    <ol>
169 *       <li>{@link BeanFilter} - Allows you to tailor handling of bean classes.
170 *          This class can be considered a programmatic equivalent to the {@link Bean} annotation when
171 *          annotating classes are not possible (e.g. you don't have access to the source).
172 *       <li>{@link PojoFilter} - Allows you to convert objects to serializable forms.
173 *    </ol>
174 * <p>
175 *    See {@link com.ibm.juno.core.filter} for more information.
176 *
177 *
178 * <h5 class='topic'>ClassMetas</h5>
179 * <p>
180 *    The {@link ClassMeta} class is a wrapper around {@link Class} object that provides cached information
181 *    about that class (e.g. whether it's a {@link Map} or {@link Collection} or bean).
182 * <p>
183 *    As a general rule, it's best to reuse bean contexts (and therefore serializers and parsers too)
184 *    whenever possible since it takes some time to populate the internal {@code ClassMeta} object cache.
185 *    By reusing bean contexts, the class type metadata only needs to be calculated once which significantly
186 *    improves performance.
187 * <p>
188 *    See {@link ClassMeta} for more information.
189 *
190 * @author Barry M. Caceres
191 * @author James Bognar (jbognar@us.ibm.com)
192 */

247   BeanContext(BeanContextFactory bc) {
248      beansRequireDefaultConstructor = bc.beansRequireDefaultConstructor;
249      beansRequireSerializable = bc.beansRequireSerializable;
250      beansRequireSettersForGetters = bc.beansRequireSettersForGetters;
251      beansRequireSomeProperties = bc.beansRequireSomeProperties;
252      beanMapPutReturnsOldValue = bc.beanMapPutReturnsOldValue;
253      useInterfaceProxies = bc.useInterfaceProxies;
254      ignoreUnknownBeanProperties = bc.ignoreUnknownBeanProperties;
255      ignoreUnknownNullBeanProperties = bc.ignoreUnknownNullBeanProperties;
256      ignorePropertiesWithoutSetters = bc.ignorePropertiesWithoutSetters;
257      ignoreInvocationExceptionsOnGetters = bc.ignoreInvocationExceptionsOnGetters;
258      ignoreInvocationExceptionsOnSetters = bc.ignoreInvocationExceptionsOnSetters;
259      useJavaBeanIntrospector = bc.useJavaBeanIntrospector;
260      beanConstructorVisibility = bc.beanConstructorVisibility;
261      beanClassVisibility = bc.beanClassVisibility;
262      beanMethodVisibility = bc.beanMethodVisibility;
263      beanFieldVisibility = bc.beanFieldVisibility;
264
265      b =
266         (beansRequireDefaultConstructor ? 1<<1 : 0)
267            + (beansRequireSerializable ? 1<<2 : 0)
268            + (beansRequireSettersForGetters ? 1<<3 : 0)
269            + (beansRequireSomeProperties ? 1<<4 : 0)
270            + (beanMapPutReturnsOldValue ? 1<<5 : 0)
271            + (useJavaBeanIntrospector ? 1<<6 : 0)
272            + (useInterfaceProxies ? 1<<8 : 0)
273            + (ignoreUnknownBeanProperties ? 1<<9 : 0)
274            + (ignoreUnknownNullBeanProperties ? 1<<10 : 0)
275            + (ignorePropertiesWithoutSetters ? 1<<11 : 0)
276            + (ignoreInvocationExceptionsOnGetters ? 1<<12 : 0)
277            + (ignoreInvocationExceptionsOnSetters ? 1<<13 : 0);
278
279      LinkedList<BeanFilter> lbf = new LinkedList<BeanFilter>();
280      LinkedList<PojoFilter> lpf = new LinkedList<PojoFilter>();
281      for (Class<?> c : bc.filters) {
282         if (isParentClass(Filter.class, c)) {
283            try {
284               if (isParentClass(BeanFilter.class, c)) {
285                  BeanFilter f = (BeanFilter)c.newInstance();
286                  f.setBeanContext(this);
287                  lbf.add(f);
288               } else if (isParentClass(PojoFilter.class, c)) {
289                  PojoFilter f = (PojoFilter)c.newInstance();
290                  f.setBeanContext(this);
291                  lpf.add(f);
292               }
293            } catch (Exception e) {
294               throw new RuntimeException(e);
295            }
296         } else {
297            if (! c.getClass().isInterface()) {
298               List<SurrogateFilter<?,?>> l = SurrogateFilter.findFilters(c);
299               if (! l.isEmpty()) {
300                  for (SurrogateFilter<?,?> f : l) {
301                     f.setBeanContext(this);
302                     lpf.add(f);
303                  }
304                  continue;
305               }
306            }
307            BeanFilter f = new InterfaceBeanFilter(c);
308            f.setBeanContext(this);
309            lbf.add(f);
310         }
311      }
312      beanFilters = lbf.toArray(new BeanFilter[0]);
313      pojoFilters = lpf.toArray(new PojoFilter[0]);
314      notBeanClasses = bc.notBeanClasses.toArray(new Class[0]);
315      notBeanPackages = bc.notBeanPackages.toArray(new String[0]);
316      implClasses.putAll(bc.implClasses);
317      uriVars.putAll(bc.uriVars);
318      implKeyClasses = implClasses.keySet().toArray(new Class[0]);
319      implValueClasses = implClasses.values().toArray(new Class[0]);
320      uriKeys = uriVars.keySet().toArray(new String[0]);
321      uriVals = uriVars.values().toArray(new String[0]);
322
323      List<String> l1 = new LinkedList<String>();
324      List<String> l2 = new LinkedList<String>();
325      for (String s : notBeanPackages) {
326         if (s.endsWith(".*"))
327            l2.add(s.substring(0, s.length()-2));
328         else
329            l1.add(s);
330      }
331      notBeanPackageNames = l1.toArray(new String[l1.size()]);
332      notBeanPackagePrefixes = l2.toArray(new String[l2.size()]);
333
334      defaultParser = bc.defaultParser;
335
336      int h = b;
337      for (BeanFilter f : beanFilters)
338         h = hash(h, f.hashCode());
339      for (PojoFilter f : pojoFilters)
340         h = hash(h, f.hashCode());
341      h = hash(h, beanConstructorVisibility.hashCode());
342      h = hash(h, beanClassVisibility.hashCode());
343      h = hash(h, beanMethodVisibility.hashCode());
344      h = hash(h, beanFieldVisibility.hashCode());
345      for (Class c : notBeanClasses)
346         h += c.hashCode();
347      for (String s : notBeanPackages)
348         h += s.hashCode();
349      for (Map.Entry<Class<?>,Class<?>> e : implClasses.entrySet()) {
350         h += e.getKey().hashCode();
351         h += e.getValue().hashCode();
352      }
353      for (Map.Entry<String,String> e : uriVars.entrySet()) {
354         h += e.getKey().hashCode();
355         h += e.getValue().hashCode();
356      }
357      hashCode = h;
358
359      Map<Class,ClassMeta> cmc = cmCacheCache.get(this);
360      if (cmc == null) {
361         cmc = new ConcurrentHashMap<Class,ClassMeta>();
362         cmc.put(Object.class, new ClassMeta(Object.class, this));
363         cmc.put(String.class, new ClassMeta(String.class, this));
364         Map<Class,ClassMeta> cmc2 = cmCacheCache.putIfAbsent(this, cmc);
365         cmc = (cmc2 == null ? cmc : cmc2);
366      }
367      cmCache = cmc;
368      cmObject = cmc.get(Object.class);
369      cmString = cmc.get(String.class);
370
371      classLoader = bc.classLoader;
372   }

397   /**
398    * Determines whether the specified class is ignored as a bean class based on the various
399    *    exclusion parameters specified on this context class.
400    *
401    * @param c The class type being tested.
402    * @return <jk>true</jk> if the specified class matches any of the exclusion parameters.
403    */
404   protected boolean isNotABean(Class<?> c) {
405      if (c.isArray() || c.isPrimitive() || c.isEnum() || c.isAnnotation())
406         return true;
407      Package p = c.getPackage();
408      if (p != null) {
409         for (String p2 : notBeanPackageNames)
410            if (p.getName().equals(p2))
411               return true;
412         for (String p2 : notBeanPackagePrefixes)
413            if (p.getName().startsWith(p2))
414               return true;
415      }
416      for (Class exclude : notBeanClasses)
417         if (isParentClass(exclude, c))
418            return true;
419      return false;
420   }

481   /**
482    * Wraps an object inside a {@link BeanMap} object (i.e.: a modifiable {@link Map})
483    * defined as a bean for one of its class, a super class, or an implemented interface.
484    * <p>
485    *    If object is not a true bean, throws a {@link BeanRuntimeException} with an explanation of why it's not a bean.
486    *
487    * <dl>
488    *    <dt>Example:</dt>
489    *    <dd>
490    * <p class='bcode'>
491    *    <jc>// Construct a bean map for new bean using only properties defined in a superclass</jc>
492    *    BeanMap&lt;MySubBean&gt; bm = BeanContext.<jsf>DEFAULT</jsf>.forBean(<jk>new</jk> MySubBean(), MySuperBean.<jk>class</jk>);
493    *
494    *    <jc>// Construct a bean map for new bean using only properties defined in an interface</jc>
495    *    BeanMap&lt;MySubBean&gt; bm = BeanContext.<jsf>DEFAULT</jsf>.forBean(<jk>new</jk> MySubBean(), MySuperInterface.<jk>class</jk>);
496    * </p>
497    *    </dd>
498    * </dl>
499    *
500    * @param <T> The class of the object being wrapped.
501    * @param o The object to wrap in a bean interface.  Must not be null.
502    * @param c The superclass to narrow the bean properties to.  Must not be null.
503    * @return The bean representation, or <jk>null</jk> if the object is not a true bean.
504    * @throws NullPointerException If either parameter is null.
505    * @throws IllegalArgumentException If the specified object is not an an instance of the specified class.
506    * @throws BeanRuntimeException If specified object is not a bean according to the bean rules
507    *       specified in this context class.
508    */
509   public <T> BeanMap<T> forBean(T o, Class<? super T> c) throws BeanRuntimeException {
510      assertFieldNotNull(o, "o");
511      assertFieldNotNull(c, "c");
512
513      if (! c.isInstance(o))
514         illegalArg("The specified object is not an instance of the specified class.  class=''{0}'', objectClass=''{1}'', object=''{2}''", c.getName(), o.getClass().getName(), 0);
515
516      ClassMeta cm = getClassMeta(c);
517
518      BeanMeta m = cm.getBeanMeta();
519      if (m == null)
520         throw new BeanRuntimeException(c, "Class is not a bean.  Reason=''{0}''", cm.getNotABeanReason());
521      return new BeanMap<T>(o, m);
522   }

547   /**
548    * Same as {@link #newBeanMap(Class)}, except used for instantiating inner member classes that must
549    *    be instantiated within another class instance.
550    *
551    * @param <T> The class of the object being wrapped.
552    * @param c The name of the class to create a new instance of.
553    * @param outer If class is a member class, this is the instance of the containing class.
554    *    Should be <jk>null</jk> if not a member class.
555    * @return A new instance of the class.
556    */
557   public <T> BeanMap<T> newBeanMap(Object outer, Class<T> c) {
558      BeanMeta m = getBeanMeta(c);
559      if (m == null)
560         return null;
561      T bean = null;
562      if (m.constructorArgs.length == 0) {

671    * Construct a {@code ClassMeta} wrapper around a {@link Class} object.
672    *
673    * @param <T> The class type being wrapped.
674    * @param c The class being wrapped.
675    *    of type {@link Class} or {@link ClassMeta}.
676    * @return If the class is not an array, returns a cached {@link ClassMeta} object.
677    *    Otherwise, returns a new {@link ClassMeta} object every time.<br>
678    */
679   public <T> ClassMeta<T> getClassMeta(Class<T> c) {
680
681      // If this is an array, then we want it wrapped in an uncached ClassMeta object.
682      // Note that if it has a pojo filter, we still want to cache it so that
683      // we can cache something like byte[] with ByteArrayBase64Filter.
684      if (c.isArray() && findPojoFilter(c) == null)
685         return new ClassMeta(c, this);
686
687      // This can happen if we have filters defined against String or Object.
688      if (cmCache == null)
689         return null;
690
691      ClassMeta<T> cm = cmCache.get(c);
692      if (cm == null) {
693
694         synchronized (this) {
695
696            // Make sure someone didn't already set it while this thread was blocked.
697            cm = cmCache.get(c);
698            if (cm == null) {
699
700               // Note:  Bean properties add the possibility that class reference loops exist.
701               // To handle this possibility, we create a set of pending ClassMetas, and
702               // call init (which finds the bean properties) after it's been added to the pending set.
703               for (ClassMeta pcm : pendingClassMetas)
704                  if (pcm.innerClass == c)
705                     return pcm;
706
707               cm = new ClassMeta<T>(c, this, true);
708               pendingClassMetas.addLast(cm);
709               try {
710                  cm.init();
711               } finally {
712                  pendingClassMetas.removeLast();
713               }
714               cmCache.put(c, cm);
715            }
716         }
717      }
718      return cm;
719   }

829   ClassMeta getClassMeta(Type o, Map<Class<?>,Class<?>[]> typeVarImpls) {
830      if (o == null)
831         return null;
832
833      if (o instanceof ClassMeta)
834         return (ClassMeta)o;
835
836      Class c = null;
837      if (o instanceof Class) {
838         c = (Class)o;
839      } else if (o instanceof ParameterizedType) {
840         // A parameter (e.g. <String>.
841         c = (Class<?>)((ParameterizedType)o).getRawType();
842      } else if (o instanceof GenericArrayType) {
843         // An array parameter (e.g. <byte[]>.
844         GenericArrayType gat = (GenericArrayType)o;
845         Type gatct = gat.getGenericComponentType();
846         if (gatct instanceof Class) {
847            Class gatctc = (Class)gatct;
848            c = Array.newInstance(gatctc, 0).getClass();
849         } else if (gatct instanceof ParameterizedType) {
850            Class gatctc = (Class<?>)((ParameterizedType)gatct).getRawType();
851            c = Array.newInstance(gatctc, 0).getClass();
852         } else {
853            return null;
854         }
855      } else if (o instanceof TypeVariable) {
856         if (typeVarImpls != null) {
857            TypeVariable t = (TypeVariable) o;
858            String varName = t.getName();
859            int varIndex = -1;
860            Class gc = (Class)t.getGenericDeclaration();
861            TypeVariable[] tv = gc.getTypeParameters();
862            for (int i = 0; i < tv.length; i++) {
863               if (tv[i].getName().equals(varName)) {
864                  varIndex = i;
865               }
866            }
867            if (varIndex != -1) {
868
869               // If we couldn't find a type variable implementation, that means
870               // the type was defined at runtime (e.g. Bean b = new Bean<Foo>();)
871               // in which case the type is lost through erasure.
872               // Assume java.lang.Object as the type.
873               if (! typeVarImpls.containsKey(gc))
874                  return object();
875
876               return getClassMeta(typeVarImpls.get(gc)[varIndex]);
877            }
878         }
879         // We don't know the bounded type, so just resolve to Object.
880         return object();
881      } else {
882         // This can happen when trying to resolve the "E getFirst()" method on LinkedList, whose type is a TypeVariable
883         // These should just resolve to Object.
884         return object();
885      }
886
887      ClassMeta rawType = getClassMeta(c);
888
889      // If this is a Map or Collection, and the parameter types aren't part
890      // of the class definition itself (e.g. class AddressBook extends List<Person>),
891      // then we need to figure out the parameters.
892      if (rawType.isMap() || rawType.isCollection()) {
893         ClassMeta[] params = findParameters(o, c);
894         if (params == null)
895            return rawType;
896         if (rawType.isMap()) {
897            if (params.length != 2)
898               return rawType;
899            if (params[0].isObject() && params[1].isObject())
900               return rawType;
901            return new ClassMeta(rawType.innerClass, this).setKeyType(params[0]).setValueType(params[1]);
902         }
903         if (rawType.isCollection()) {
904            if (params.length != 1)
905               return rawType;
906            if (params[0].isObject())
907               return rawType;
908            return new ClassMeta(rawType.innerClass, this).setElementType(params[0]);
909         }
910      }
911
912      return rawType;
913   }

930   ClassMeta[] findParameters(Type o, Class c) {
931      if (o == null)
932         o = c;
933
934      // Loop until we find a ParameterizedType
935      if (! (o instanceof ParameterizedType)) {
936         loop: do {
937            o = c.getGenericSuperclass();
938            if (o instanceof ParameterizedType)
939               break loop;
940            for (Type t : c.getGenericInterfaces()) {
941               o = t;
942               if (o instanceof ParameterizedType)
943                  break loop;
944            }
945            c = c.getSuperclass();
946         } while (c != null);
947      }
948
949      if (o instanceof ParameterizedType) {
950         ParameterizedType pt = (ParameterizedType)o;
951         if (! pt.getRawType().equals(Enum.class)) {
952            List<ClassMeta<?>> l = new LinkedList<ClassMeta<?>>();
953            for (Type pt2 : pt.getActualTypeArguments()) {
954               if (pt2 instanceof WildcardType || pt2 instanceof TypeVariable)
955                  return null;
956               l.add(getClassMeta(pt2, null));
957            }
958            if (l.isEmpty())
959               return null;
960            return l.toArray(new ClassMeta[l.size()]);
961         }
962      }
963
964      return null;
965   }

1039   /**
1040    * Converts class name strings to ClassMeta objects.
1041    *
1042    * <dl>
1043    *    <dt>Example:</dt>
1044    *    <dd>
1045    * <ul>
1046    *    <li>{@code java.lang.String}
1047    *    <li>{@code com.ibm.sample.MyBean[]}
1048    *    <li>{@code java.util.HashMap<java.lang.String,java.lang.Integer>}
1049    *    <li>{@code [Ljava.lang.String;} (i.e. the value of <code>String[].<jk>class</jk>.getName()</code>
1050    * </ul>
1051    *    </dd>
1052    * </dl>
1053    *
1054    * @param s The class name.
1055    * @return The ClassMeta corresponding to the class name string.
1056    */
1057   public ClassMeta<?> getClassMetaFromString(String s) {
1058      int d = 0;
1059      if (s == null || s.isEmpty())
1060         return null;
1061
1062      // Check for Class.getName() on array class types.
1063      if (s.charAt(0) == '[') {
1064         try {
1065            return getClassMeta(findClass(s));
1066         } catch (ClassNotFoundException e) {
1067            throw new RuntimeException(e);
1068         }
1069      }
1070
1071      int i1 = 0;
1072      int i2 = 0;
1073      int dim = 0;
1074      List<ClassMeta<?>> p = null;
1075      for (int i = 0; i < s.length(); i++) {
1076         char c = s.charAt(i);
1077         if (c == '<') {
1078            if (d == 0) {
1079               i1 = i;
1080               i2 = i+1;
1081               p = new LinkedList<ClassMeta<?>>();
1082            }
1083            d++;
1084         } else if (c == '>') {
1085            d--;
1086            if (d == 0 && p != null)
1087               p.add(getClassMetaFromString(s.substring(i2, i)));
1088         } else if (c == ',' && d == 1) {
1089            if (p != null)
1090               p.add(getClassMetaFromString(s.substring(i2, i)));
1091            i2 = i+1;
1092         }
1093         if (c == '[') {
1094            if (i1 == 0)
1095               i1 = i;
1096            dim++;
1097         }
1098      }
1099      if (i1 == 0)
1100         i1 = s.length();
1101      try {
1102         String name = s.substring(0, i1).trim();
1103         char x = name.charAt(0);
1104         Class<?> c = null;
1105         if (x >= 'b' && x <= 's') {
1106            if (x == 'b' && name.equals("boolean"))
1107               c = boolean.class;
1108            else if (x == 'b' && name.equals("byte"))
1109               c = byte.class;
1110            else if (x == 'c' && name.equals("char"))
1111               c = char.class;
1112            else if (x == 'd' && name.equals("double"))
1113               c = double.class;
1114            else if (x == 'i' && name.equals("int"))
1115               c = int.class;
1116            else if (x == 'l' && name.equals("long"))
1117               c = long.class;
1118            else if (x == 's' && name.equals("short"))
1119               c = short.class;
1120            else
1121               c = findClass(name);
1122         } else {
1123            c = findClass(name);
1124         }
1125
1126         ClassMeta<?> cm = getClassMeta(c);
1127
1128         if (p != null) {
1129            if (cm.isMap())
1130               cm = new ClassMeta(c, this).setKeyType(p.get(0)).setValueType(p.get(1));
1131            if (cm.isCollection())
1132               cm = new ClassMeta(c, this).setElementType(p.get(0));
1133         }
1134
1135         while (dim > 0) {
1136            cm = new ClassMeta(Array.newInstance(cm.getInnerClass(), 0).getClass(), this);
1137            dim--;
1138         }
1139
1140         return cm;
1141      } catch (ClassNotFoundException e) {
1142         throw new RuntimeException(e);
1143      }
1144   }

1150   /**
1151    * Returns the {@link PojoFilter} associated with the specified class, or <jk>null</jk> if there is no
1152    * pojo filter associated with the class.

1440         if (value == null) {
1441

1443            if (type.isPrimitive())

1447            return null;
1448         }

1514            } else if (type.isBoolean()) {

1521         }

1523         if (type.isNumber()) {
