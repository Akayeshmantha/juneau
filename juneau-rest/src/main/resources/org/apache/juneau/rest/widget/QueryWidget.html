query
Search (?): Comma-delimited list of key/value pair search terms.
Keys are column names.
Values are search terms.
Refer to the Queryable javadocs for a full explanation of possible search terms.
Example: column1=foo*, column2<100, column3=2013-2016.06.30
View (?): Comma-delimited list of columns to display.
Example: column1, column2
Sort (?): Comma-delimited list of columns to sort by.
Columns can be suffixed with '-' to indicate descending order.
Example: column1, column2-
