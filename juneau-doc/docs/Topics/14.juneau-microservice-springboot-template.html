Starter Project Zip

my-microservice-springboot-{@property juneauVersion}.zip

The my-microservice-springboot.zip file is a predefined starter Eclipse project for developing REST microservices using the {@doc juneau-rest-server-springboot} package with Spring Boot.

It includes a combination of the Juneau Core, Server, and Client APIs and all libraries needed to execute in a Java 1.8+ environment.
