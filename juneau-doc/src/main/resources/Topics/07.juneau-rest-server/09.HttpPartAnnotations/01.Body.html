The {@link org.apache.juneau.http.annotation.Body @Body} annotation is used to identify POJOs to be used as the body of an HTTP request.

  • {@link org.apache.juneau.http.annotation.Body Body}
    • {@link org.apache.juneau.http.annotation.Body#partSerializer() partSerializer()} - Override the part serializer.
    • {@link org.apache.juneau.http.annotation.Body#partParser() partParser()} - Override the part parser.
    • {@link org.apache.juneau.http.annotation.Body#required() required()} - Input validation. Body must be present.
    • {@link org.apache.juneau.http.annotation.Body#schema() schema()} - The schema.

This annotation can be applied at the following locations:

  • Parameters on a @RestMethod-annotated method.
  • POJO classes used as parameters on a @RestMethod-annotated method.
Examples:

// Defined on parameter @RestMethod(name=POST) public void addPerson(@Body Person person) {...}

// Defined on POJO class @RestMethod(name=POST) public void addPerson(Person person) {...} @Body public class Person {...}

This is functionally equivalent to the following code...

@RestMethod(name=POST) public void addPerson(RestRequest req) { Person person = req.getBody().asType(Person.class); ... }

Examples:

// POJO convertible from string @RestMethod(name=PUT, path="/echoPojo") public MyPojo echo(@Body MyPojo p) { return p; } public static class MyPojo { private String val; public MyPojo(String val) { this.val = val; } public String toString() { return val; } }

// Parsable bean @RestMethod(name=PUT, path="/echoBean") public MyBean echo(@Body MyBean b) { return b; } public static class MyBean { public String f1; }

// List of parsable beans @RestMethod(name=PUT, path="/echoBeanList") public MyBeanList echo(@Body MyBeanList l) { return l; } public static class MyBeanList extends LinkedList<MyBean> {}

// POJO convertible from an input stream @RestMethod(name=PUT, path="/echoPojo") public MyPojo echo(@Body MyPojo p) { return p; } public static class MyPojo { private String val; public MyPojo(InputStream is) { this.val = IOUtils.read(is); } public String toString() { return val; } }

This annotation is also used for supplying swagger information about the body of the request.
This information is used to populate the auto-generated Swagger documentation and UI.

Examples:

// Normal @Body( description="Pet object to add to the store", required=true, example="{name:'Doggie',price:9.99,species:'Dog',tags:['friendly','cute']}" )

// Free-form @Body({ "description: 'Pet object to add to the store',", "required: true,", "example: {name:'Doggie',price:9.99,species:'Dog',tags:['friendly','cute']}," "x-extra: 'extra field'" })

// Localized @Body( description="$L{My.Localized.Description}", required=true, example="{name:'Doggie',price:9.99,species:'Dog',tags:['friendly','cute']}" )

Several of the attributes on the @Body annotation are used for specifying additional information for the auto-generated Swagger documentation.
These do not have any affects on the running code except for the generated Swagger:

  • {@link org.apache.juneau.http.annotation.Body}
    • {@link org.apache.juneau.http.annotation.Body#api() api()} - Free-form JSON swagger.
    • {@link org.apache.juneau.http.annotation.Body#description() description()} - Description
    • {@link org.apache.juneau.http.annotation.Body#example() example()} - Defines a serialized POJO example (in JSON format) used to populate examples per language type.
    • {@link org.apache.juneau.http.annotation.Body#examples() examples()} - Map of examples per language type.
    • {@link org.apache.juneau.http.annotation.Body#schema() schema()} - The swagger schema ({@link org.apache.juneau.http.annotation.Schema Schema}).
  • {@link org.apache.juneau.http.annotation.Schema Schema}
    • {@link org.apache.juneau.http.annotation.Schema#$ref() $ref()} - Reference to schema definition.
    • {@link org.apache.juneau.http.annotation.Schema#description() description()} - Description
    • {@link org.apache.juneau.http.annotation.Schema#example() example()} - Defines a serialized POJO example (in JSON format) used to populate examples per language type.
    • {@link org.apache.juneau.http.annotation.Schema#examples() examples()} - Map of examples per language type.
    • {@link org.apache.juneau.http.annotation.Schema#externalDocs() externalDocs()} - External documentation.
    • {@link org.apache.juneau.http.annotation.Schema#title() title()} - Title.
    • {@link org.apache.juneau.http.annotation.Schema#value() value()} - Free-form JSON swagger.
    • {@link org.apache.juneau.http.annotation.Schema#xml() xml()} - XML schema.
  • {@link org.apache.juneau.http.annotation.Items Items}
    • {@link org.apache.juneau.http.annotation.Items#$ref() $ref()} - Reference to schema definition.
    • {@link org.apache.juneau.http.annotation.Items#value() value()} - Free-form JSON swagger.

Other Notes:
  • Annotation parameter values will be aggregated when used on POJO parent and child classes.
    Values on child classes override values on parent classes.
  • Annotation parameter values will be aggregated when used on both POJOs and REST methods.
    Values on methods override values on POJO classes.
