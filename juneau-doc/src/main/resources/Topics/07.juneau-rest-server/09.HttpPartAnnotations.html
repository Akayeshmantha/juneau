{todo} HTTP-Part Annotations

The annotations used for defining the schema for an HTTP part are:

  • {@link org.apache.juneau.http.annotation.Body Body}
  • {@link org.apache.juneau.http.annotation.Header Header}
  • {@link org.apache.juneau.http.annotation.FormData FormData}
  • {@link org.apache.juneau.http.annotation.Query Query}
  • {@link org.apache.juneau.http.annotation.Path Path}

The sub-annotations used in the annotation above are:

  • {@link org.apache.juneau.http.annotation.Schema Schema}
    • {@link org.apache.juneau.http.annotation.Schema#_default() _default()} - Default value if not present.
    • {@link org.apache.juneau.http.annotation.Schema#_enum() _enum()} - Input validation. Must match one of the values.
    • {@link org.apache.juneau.http.annotation.Schema#exclusiveMaximum() exclusiveMaximum()} - Input validation. Whether maximum is exclusive.
    • {@link org.apache.juneau.http.annotation.Schema#exclusiveMinimum() exclusiveMinimum()} - Input validation. Whether minimum is exclusive.
    • {@link org.apache.juneau.http.annotation.Schema#format() format()} - The schema type format.
    • {@link org.apache.juneau.http.annotation.Schema#items() items()} - The schema of items in a collection.
    • {@link org.apache.juneau.http.annotation.Schema#maximum() maximum()} - Input validation. Maximum numeric value.
    • {@link org.apache.juneau.http.annotation.Schema#maxItems() maxItems()} - Input validation. Maximum number of items in a collection.
    • {@link org.apache.juneau.http.annotation.Schema#maxLength() maxLength()} - Input validation. Maximum length of a string.
    • {@link org.apache.juneau.http.annotation.Schema#minimum() minimum()} - Input validation. Minimum numeric value.
    • {@link org.apache.juneau.http.annotation.Schema#minItems() minItems()} - Input validation. Minimum number of items in a collection.
    • {@link org.apache.juneau.http.annotation.Schema#minLength() minLength()} - Input validation. Minimum length of a string.
    • {@link org.apache.juneau.http.annotation.Schema#multipleOf() multipleOf()} - Input validation. Number must be a multiple of.
    • {@link org.apache.juneau.http.annotation.Schema#pattern() pattern()} - Input validation. Must match regular expression.
    • {@link org.apache.juneau.http.annotation.Schema#pattern() properties()} - Schema of properties of this item.
    • {@link org.apache.juneau.http.annotation.Schema#required() required()} - Whether this value is required.
    • {@link org.apache.juneau.http.annotation.Schema#type() type()} - The schema type.
    • {@link org.apache.juneau.http.annotation.Schema#uniqueItems() uniqueItems()} - Input validation. Collections must contain unique items only.
  • {@link org.apache.juneau.http.annotation.Items Items}
    • {@link org.apache.juneau.http.annotation.Items#_default() _default()} - Default value if not present.
    • {@link org.apache.juneau.http.annotation.Items#_enum() _enum()} - Input validation. Must match one of the values.
    • {@link org.apache.juneau.http.annotation.Items#collectionFormat() collectionFormat()} - How collections of items are formatted.
    • {@link org.apache.juneau.http.annotation.Items#exclusiveMaximum() exclusiveMaximum()} - Input validation. Whether maximum is exclusive.
    • {@link org.apache.juneau.http.annotation.Items#exclusiveMinimum() exclusiveMinimum()} - Input validation. Whether minimum is exclusive.
    • {@link org.apache.juneau.http.annotation.Items#format() format()} - The schema type format.
    • {@link org.apache.juneau.http.annotation.Items#items() items()} - The schema of items in a collection.
    • {@link org.apache.juneau.http.annotation.Items#maximum() maximum()} - Input validation. Maximum numeric value.
    • {@link org.apache.juneau.http.annotation.Items#maxItems() maxItems()} - Input validation. Maximum number of items in a collection.
    • {@link org.apache.juneau.http.annotation.Items#maxLength() maxLength()} - Input validation. Maximum length of a string.
    • {@link org.apache.juneau.http.annotation.Items#minimum() minimum()} - Input validation. Minimum numeric value.
    • {@link org.apache.juneau.http.annotation.Items#minItems() minItems()} - Input validation. Minimum number of items in a collection.
    • {@link org.apache.juneau.http.annotation.Items#minLength() minLength()} - Input validation. Minimum length of a string.
    • {@link org.apache.juneau.http.annotation.Items#multipleOf() multipleOf()} - Input validation. Number must be a multiple of.
    • {@link org.apache.juneau.http.annotation.Items#pattern() pattern()} - Input validation. Must match regular expression.
    • {@link org.apache.juneau.http.annotation.Items#type() type()} - The schema type.
    • {@link org.apache.juneau.http.annotation.Items#uniqueItems() uniqueItems()} - Input validation. Collections must contain unique items only.
Other Notes:
  • Annotation parameter values will be aggregated when used on POJO parent and child classes.
    Values on child classes override values on parent classes.
  • Annotation parameter values will be aggregated when used on both POJOs and REST methods.
    Values on methods override values on POJO classes.
