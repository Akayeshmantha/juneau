{new} HTTP-Part Annotations

The Java method arguments on 3rd-party REST interfaces can be annotated with any of the following:

  • {@link org.apache.juneau.http.annotation.Body} - The HTTP request body.
  • {@link org.apache.juneau.http.annotation.Query} - A URL query parameter.
  • {@link org.apache.juneau.http.annotation.FormData} - A form-data parameter.
  • {@link org.apache.juneau.http.annotation.Header} - A request header.
  • {@link org.apache.juneau.http.annotation.Path} - A request path variable.
  • {@link org.apache.juneau.http.annotation.Request} - A request bean.

These are the exact same annotations used for server-side HTTP parts.
